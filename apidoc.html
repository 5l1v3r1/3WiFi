Описание API

Сервис предоставляет доступ к базе данных через API.
Ниже описан процесс посылки запроса, а так же возможные типы запросов.

Процесс посылки запроса:
Обработчик находится по адресу: "/api/ajax.php". В для авторизации используются API-ключи. В настоящий момент реализован только функционал для R-ключей.
Методом GET передаются параметры "Query" (название вызываемого API) и "Key" (R или W ключ, в зависимости от вызываемого API).
Возможные значение параметра Query:

GetApiKeys
Пример: Query=GetApiKeys
Метод: POST Параметр Login - логин пользователя
Параметр Password - пароль пользователя
Возвращается ответ в виде JSON-объекта вида: {"Successes":true, "lastupdate": *unixtime*, "r": *r-key*, "w": *w-key*}
